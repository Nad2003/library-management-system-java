# 📚 Library Management System (Java + JDBC)

## 🚀 Overview

This is a **console-based Library Management System** developed using **Java and JDBC with MySQL**.
The project simulates real-world library operations with both **Student** and **Admin** functionalities.

---

## ✨ Features

### 👤 Student Module

* Register new student
* Login with credentials
* Issue book
* Return book
* View available books

---

### 👨‍💼 Admin Module

* Admin login
* Add new books
* Update book quantity
* View all books
* Manage library inventory

---

## 🧠 Key Concepts Used

* Java (Core + OOP)
* JDBC (Database Connectivity)
* MySQL (Relational Database)
* PreparedStatement (SQL security)
* Menu-driven console application

---

## 🗄️ Database Tables

* **STUDENT** → Stores student details
* **BOOKS** → Stores book information
* **ISSUE** → Tracks issued books
* **ADMIN** → Admin credentials

---

## ⚙️ How It Works

1. User selects **Register / Login / Admin Login**
2. Students can issue or return books
3. Admin manages books and system data
4. Database keeps track of all operations
